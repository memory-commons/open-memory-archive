# Security Policy

## Reporting

Please report security issues privately to the project maintainers. Do not open a public issue for vulnerabilities involving private data exposure, archive tampering, path traversal, or unsafe file handling.

## Supported Versions

The project is pre-1.0. Security fixes target the latest release.

## Scope

Relevant issues include:

- generated archives including private source paths or signed URLs
- redaction bypasses
- path traversal in media copy handling
- unexpected network calls in the builder or viewer
- checksum generation errors
